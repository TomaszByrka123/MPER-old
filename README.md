Usage
====================

Clone repository:

    git clone https://github.com/TomaszByrka123/lazik-szkola

Creating virtual environment:

    cd lazik-szkola/raspberry/
    python3 -m venv venv

Activate virtual environment:

    source venv/bin/activate

Install requirement:

    pip3 install -r requirement.txt

Run app:

    python3 main.py
